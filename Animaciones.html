Primer Vídeo

Este vídeo consta de una animación que fue creada en base a un modelo 3D de una araña, creada desde cero, para así, poder entender como es que se puede crear un modelo 3D, tomando como referencia el movimiento de una araña, pero adaptado en el programa que utilizamos.

Segundo Vídeo

En este vídeo, se muestra la animación de un globo, que fue hecho dentro del programa Blender, con el fin de poder mostrar todas las capacidades que tenemos de manejar el programa y crear diferentes escenarios y perspectivas de manera creativa e innovadora.

Tercer Vídeo

Para la creación de este vídeo, se tuvo que tomar otra perspectiva de la finalización del vídeo, así que se decidio hacer la imitación de una nave espacial, añadiendo los frames necesarios para hacer la escena muy realista, mostrando como resultado final el vídeo que se muestra a continuación.

Cuarto Vídeo

Este vídeo es un ejemplo de como hacer una animación simple, sin hacer mucho y que quede de manera muy agradable y enfocada a la idea del vídeo, con las simples expresiones de la taza de cafe, podemos mostrar simplicidad y creatividad en la animación. mostrando lo que podemos hacer, para entregar trabajos de calidad.
